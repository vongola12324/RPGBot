# RPGBot
RPG + 養成 + Plurk Bot + FB Bot + Twitter Bot + 網站 => ???

## Requirement
- Python 3
- Django

## Deployment
1. Install Python3
2. Install Python package
```shell=
pip3 install -r requirement.txt
```
3. Sync DB
```shell=
python3 manage.py migrate
```
4. Run Server (port 8000)
```shell=
python3 manage.py runserver 8000
```
